Word Prediction Algorithm

Data Science Specialization Capstone Project

Jesse Peterson-Brandt

My Application

My word prediction app takes a phrase of any length and predicts the next word using a back-off model. It begins by using up to four words and, if it does not find that exact phrase, works down to predicting based on just the last word.

Data

Data were provided by the Coursera Data Science Specialization and Swiftkey. They included text from Twitter, news websites and blogs. I used this data to train my model and make predictions.

Tokenization

First we tokenize the data. I tokenized different n-grams separately. These are bigrams.

tweetTokens <- NGramTokenizer(twitter, control)

Convert to Data Frame

The last thing was to convert the counts to a data frame for easier searching. I had help in this from StackOverflow: http://stackoverflow.com/questions/7069076/split-column-at-delimiter-in-data-frame

tweetDF <- data.frame(do.call('rbind', strsplit(as.character(tweetCount$x),' ',fixed=TRUE)))
tweetDF <- cbind(tweetDF, tweetCount$freq)
